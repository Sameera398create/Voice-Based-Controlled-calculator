# Voice-Based Controlled Calculator

## Overview
This project implements a **Voice-Based Controlled Calculator** using Python.  
The system allows users to perform mathematical calculations using voice commands.
The program converts speech into text, evaluates the mathematical expression, and produces the result using text-to-speech

## Features
✔ Voice input using microphone  
✔ Speech-to-text conversion  
✔ Supports basic arithmetic operations  
✔ Text-to-speech output 

## Technologies Used
- Python
- SpeechRecognition
- PyAudio
- pyttsx3

## Example

Voice Input:
five plus three

Output:
Result: 8

## Installation

Install required libraries:
pip install SpeechRecognition pyttsx3 pyaudio
